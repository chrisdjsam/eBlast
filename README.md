# eBlast
Compassion Network - eBlast Page

### eBlast ###
The builds for this project are tested using travis:


eBlast is an AngularJS application. It shows off the following technologies:

 - AngularJS
     - directives
     - services
     - $ajax
     - $q promises
 - Compassion Network API's
 - Bootstrap
 - Less CSS
 - Jasmine Unit Testing
 - Karma test runner
 - Grunt task runner
